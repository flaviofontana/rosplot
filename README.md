rosplot
=============

Tool to plot data from ros. Derived from the rqt_plot. 
* autocompletion works with namespaces
* uses less cpu than matlibplot
* xaxis shows time
* zoom fixed
* rad2deg conversion 


##instalation
to install the gui you have to clear your rqt cache
```
rm ~/.config/ros.org/rqt_gui.ini
```
